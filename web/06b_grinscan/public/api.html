Public REST API

All endpoints return JSON. /rest/ routes include Access-Control-Allow-Origin: * for browser use. No authentication required.

Endpoint Description Cache
GET /api/stats Latest tip, hashrate, difficulty, peers, price, pool size, node mode, DB size live
GET /api/blocks Paginated block list — ?limit=N&offset=M (max 100) live
GET /api/block/:id Single block by height or hash — includes _prev_timestamp live
GET /api/history Historical block samples for charts — ?days=N (max 90). Returns { ok, rows } live
GET /api/peers Connected peer list (addr, direction, user_agent) live
GET /api/price GRIN price (USD + BTC), 24h change, price history 2 min
GET /api/tip Current tip height + hash live
GET /events Server-Sent Events stream — fires { type:"block", height } on each new block streaming
GET /rest/stats.json Core chain stats (CORS-enabled public snapshot) 30s
GET /rest/supply.json Circulating supply = height × 60 GRIN 30s
GET /rest/height.json Block height only 30s
GET /rest/difficulty.json Network difficulty + hashrate (GPS) 30s
GET /rest/emission.json Static emission schedule — yearly milestones, no halving 24h
GET /rest/node.json Connected peers, version distribution (CORS-enabled) 30s
GET /rest/price.json Price data (gate.io + nonlogs.io) + 24h change + history (CORS-enabled) 2 min

SSE Example

const es = new EventSource('/events');
es.onmessage = (e) => {
  const msg = JSON.parse(e.data);
  if (msg.type === 'block') console.log('New block:', msg.height);
};
