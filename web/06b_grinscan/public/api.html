API Reference

All endpoints return JSON. CORS-tagged routes include Access-Control-Allow-Origin: * for cross-origin browser use. No authentication required.

SSE Example

const es = new EventSource('/events');
es.onmessage = (e) => {
  const msg = JSON.parse(e.data);
  if (msg.type === 'block') console.log('New block:', msg.height);
};

GrinScan provides a public REST API for the Grin (GRIN) MimbleWimble blockchain — live block data, network hashrate, mining difficulty, GRIN/BTC price, circulating supply, peer nodes, and Server-Sent Events push notifications. Free to use, no API key required.
