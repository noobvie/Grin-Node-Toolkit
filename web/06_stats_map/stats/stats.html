Public JSON API

Free to use · CORS enabled · No authentication required · tap row to expand

GET
/api/summary ▶
Current block height, network hashrate (GPS), mining difficulty, coin supply, inflation rate
GET
/api/hashrate ▶
Network hashrate (GPS) — daily / hourly / recent arrays of [timestamp, value] pairs
GET
/api/difficulty ▶
Network mining difficulty — daily / hourly / recent history
GET
/api/transactions ▶
Transactions per block — daily / hourly / recent history
GET
/api/fees ▶
Total fees per block (nGrin) — daily / hourly / recent history
GET
/api/active_peers ▶
Active peer count by network — {"mainnet": {daily, hourly, recent}, "testnet": {…}} — each array contains [timestamp, count] pairs, sampled every ~5 min
GET
/api/versions ▶
Node version distribution — label + count for each Grin client version seen in the last 7 days
GET
/api/price ▶
GRIN/USDT (Gate.io) — current price, 24h stats, OHLCV history for Day / Week / Month / Year / All Time
GET
/api/inflation ▶
Grin annual supply inflation rate since mainnet (2019→present) — deterministic 1 GRIN/sec constant emission. Each entry: {year, supply_start, new_coins, rate_pct}
Rate limit: 30 req / min per IP  ·  Cache: 5 min  ·  Format: JSON
All endpoints return Access-Control-Allow-Origin: * — safe to fetch from any browser or app.
