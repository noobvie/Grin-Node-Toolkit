Announcements

Site notices and maintenance banners

Maintenance Mode

Shows a branded full-page notice to visitors. The admin panel stays accessible and miners keep mining (stratum is unaffected). Login / dashboard / account pages remain reachable.

Announcement Banners

Short messages shown site-wide at the top of every public page. Each banner has a type, optional link, on/off toggle, optional dismiss button, and an optional start/end date.
