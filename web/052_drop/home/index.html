About Grin Drop

Grin Drop is an open-source GRIN faucet that teaches you how real Grin transactions work. Unlike most cryptocurrencies, Grin uses interactive transactions (slatepacks) — both parties must participate to complete a transfer. This is the same flow used by mining pools and exchanges.

The testnet drop is a risk-free sandbox: claim tGRIN, run through the full 3-step slatepack flow, and understand Grin before using real funds.
